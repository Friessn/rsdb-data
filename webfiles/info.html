Installation of R-Package for connection to RSDB server

from remote (GitHub) source: (recommended, dependencies will be installed automatically)

if(!require("remotes")) install.packages("remotes")

remotes::install_github("environmentalinformatics-marburg/rsdb/r-package")

library(RSDB)

??RSDB

from local package:

Download the package "RSDB_X.tar.gz" from this folder and install in R.

install.packages("path_to_package/RSDB_X.tar.gz", repos=NULL, type="source")

installation dependencies for RSDB:

  • R6
  • httr
  • jsonlite
  • openssl
  • raster
  • rgdal

optional dependencies:

  • rgl (for visualisation in rgl::plot3d, RSDB::visualise_raster)
  • lidR (for LiDAR processing in R)
  • rlas (for LiDAR processing in R)
  • data.table (for LiDAR processing in R)
  • hsdar (Hyperspectral processing in R)
  • mapview (for visualisation in mapview::cubeview, mapview::plainview)
